	JSON, o Javascript Object Notation, es un formato de texto que es independiente del lenguaje Javascript, este utiliza convenciones que son conocidos por los programadores de la familia de lenguajes C.

**Ex.**

```json
{
	"rojo":"#f00",
	"verde":"#0f0",
	"azul":"#00f",
	"cyan":"#0ff",
	"magenta":"#f0f",
	"amarillo":"#ff0",
	"negro":"#000"
}
```
