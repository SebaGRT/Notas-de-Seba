	Tipo de dato provisto por un lenguaje de programación.

Un [[Dato|dato]] de tipo primitiva define el tipo de contenido que almacenará una variable.

**Ex.
```c
int a; // define que la variable "a" contendrá solo números enteros.
```


Cuando una variable se ve declarada como un tipo de primitiva, se reserva en la[[Memoria RAM (Random Access Memory)| memoria]] por el tamaño predefinido dependiendo del tipo de primitiva.
La sección en la memoria queda referenciada por una dirección que permite su acceso directo.

| Datos Primitivos | Byte |
| ---------------- | ---- |
| char             | 1    |
| shor int         | 2    |
| int              | 4    |
| long int         | 4    |
| bool             | 1    |
| float            | 4    |
| double           | 8    |
| long double      | 12   |
