	MQTT, o Message Queuing Telemetry Transport, es un protocolo de mensajería 
	ligero para usar. Se utiliza principalmente para comunicaciones de máquina 
	a máquina (M2M) o conexiones del tipo Internet de las cosas (IoT).
	Este protocolo emplea mensajes como elemento fundamental para comunicar 
	dichos mensajes.
